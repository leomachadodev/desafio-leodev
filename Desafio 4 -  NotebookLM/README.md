# 📚 Desafio DIO | NotebookLM como Ferramenta de Aprendizagem

## Sobre o Projeto

Este repositório foi criado como parte de um desafio prático da DIO com o objetivo de conhecer o **NotebookLM**, ferramenta de Inteligência Artificial desenvolvida pela Google para auxiliar na organização de estudos, pesquisa e geração de conhecimento a partir de diferentes fontes.

A proposta do desafio foi utilizar a IA como apoio ao aprendizado, explorando seus recursos para resumir conteúdos, responder perguntas, organizar informações e criar um material de consulta que possa ser reutilizado futuramente.

---

# Objetivos

Durante este desafio busquei:

* Conhecer os principais recursos do NotebookLM;
* Entender como a ferramenta pode auxiliar nos estudos;
* Testar diferentes estratégias de prompts;
* Organizar as informações de forma estruturada;
* Criar um pequeno guia de revisão sobre o tema estudado.

---

# O que percebi durante os testes

Uma das partes mais interessantes deste desafio foi perceber como pequenas mudanças no prompt influenciam diretamente a resposta da IA.

Algumas observações:

* perguntas muito genéricas geram respostas mais superficiais;
* prompts objetivos produzem respostas mais completas;
* dividir uma pergunta grande em partes facilita o entendimento;
* solicitar exemplos práticos torna o conteúdo mais fácil de compreender.

# Miniguia de Estudos

## O que é o NotebookLM?

O NotebookLM é uma ferramenta de Inteligência Artificial criada pela Google para apoiar estudos e pesquisas. Ela permite utilizar documentos como base de conhecimento para responder perguntas, criar resumos, comparar informações e organizar conteúdos.

---

## Principais Recursos

* Organização de documentos
* Resumos automáticos
* Perguntas e respostas
* Comparação entre fontes
* Guias de estudo
* Mapas mentais

---

# Glossário

### IA Generativa

Tecnologia capaz de criar conteúdos como textos, imagens, códigos e outros materiais utilizando modelos de Inteligência Artificial.

### LLM

Modelo de linguagem utilizado para compreender e gerar textos em linguagem natural.

### Prompt

Instrução enviada para a IA realizar determinada tarefa.

### Prompt Engineering

Processo de elaboração de prompts mais claros e específicos para obter melhores respostas.

### AI Agent

Sistema baseado em Inteligência Artificial capaz de interpretar informações, utilizar ferramentas e executar tarefas de maneira autônoma.

# Prompts que podem ser reutilizados

### Resumo

```
Resuma este conteúdo destacando apenas os principais conceitos.
```

### Comparação

```
Compare este documento com as demais fontes utilizadas.
```

### Revisão

```
Quais conceitos preciso revisar para entender melhor este assunto?
```

### Quiz

```
Crie um questionário com 10 perguntas e respostas comentadas sobre este tema.
```

### Mapa Mental

```
Organize este conteúdo em formato de mapa mental.
```

# Aprendizados

Este desafio mostrou que ferramentas de IA podem ser utilizadas como apoio ao aprendizado e não apenas para gerar respostas prontas.

Também ficou evidente que a qualidade das respostas depende da forma como a pergunta é construída. Quanto mais claro e específico é o prompt, melhores tendem a ser os resultados.

Além de conhecer o NotebookLM, o desafio reforçou a importância da organização do conhecimento e da Engenharia de Prompts para tornar o processo de estudo mais eficiente.
