# 🏎️ Desafio 6 - Corridas do Mario Kart com Node.js

Projeto desenvolvido durante a Formação Node.js da Digital Innovation One (DIO).

## 📖 Sobre o projeto

Este desafio consiste na construção de um simulador de corrida inspirado no universo Mario Kart utilizando JavaScript e Node.js.

Durante a corrida, dois personagens competem em diferentes tipos de pistas. Cada rodada sorteia um cenário diferente, exigindo atributos específicos dos competidores.

O vencedor é definido pela soma entre o resultado do dado e os atributos do personagem.

---

## 🎯 Objetivos

Aplicar conceitos de:

- Variáveis
- Objetos
- Funções
- Async / Await
- Estruturas condicionais
- Switch Case
- Loops
- Lógica de programação

---

## 🏁 Funcionamento

A corrida possui 5 rodadas.

Em cada rodada é sorteado um tipo de pista:

- 🛣️ Reta
- ↪️ Curva
- 🥊 Confronto

Cada personagem possui atributos diferentes:

- Velocidade
- Manobrabilidade
- Poder

O resultado do dado é somado ao atributo correspondente.

Ao final da corrida vence quem possuir mais pontos.


```bash

```

