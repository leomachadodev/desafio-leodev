# 🛒 Desafio 7 - Carrinho de Compras Shopee

Projeto desenvolvido durante a Formação Node.js da Digital Innovation One (DIO).

## 📖 Objetivo

Desenvolver um sistema de carrinho de compras inspirado no funcionamento da Shopee, utilizando JavaScript e Node.js.

O projeto simula as principais operações de um carrinho de compras, permitindo adicionar produtos, remover itens, excluir produtos e calcular automaticamente o valor total da compra.

---

##  Funcionalidades

- Adicionar produtos ao carrinho.
- Exibir os itens do carrinho.
- Calcular automaticamente o valor total da compra.
- Remover uma unidade de um produto.
- Excluir um produto do carrinho.

---

## Tecnologias

- JavaScript (ES Modules)
- Node.js

---

##  Estrutura do Projeto

```text
src
│
├── index.js
│
└── services
    ├── cart.js
    └── items.js
```

### Responsabilidades

**cart.js**

Responsável pelas regras de negócio do carrinho.

- Adicionar itens
- Remover itens
- Excluir produtos
- Calcular o total
- Exibir o carrinho

**items.js**

Responsável pela criação dos produtos utilizados pelo carrinho.
